## 🎯 关于本站

AINative是一个**双模式静态文档网站**，旨在通过创新的内容呈现方式，帮助企业和个人理解和实践AI Native转型。

### 双模式体验

- **📖 Document模式**：传统文档交互，支持多级导航、全文搜索、响应式布局
- **🎤 PPT模式**：基于reveal.js的演示文稿，适合团队分享和培训场景

### 核心内容

1. **AI Native简述** - 理解AI Native的本质和价值
2. **真实案例** - 行业最佳实践和成功案例
3. **转化路程** - 企业AI转型的实施路径

---

**提示：** 本站基于VuePress 2构建，支持一键切换Document和PPT两种模式。

::: tip 快速开始
点击上方"开始阅读"按钮进入Document模式，或点击"查看演示"体验PPT模式。
:::
